# Calculator Web Application

A modern calculator built using HTML, CSS, and JavaScript.
It provides a clean user interface and supports basic arithmetic operations with smooth interactions.

---

## Live Demo

https://aamir-vakot.github.io/calculator/

---

## Features

* Clean and responsive user interface
* Supports basic arithmetic operations
* Interactive buttons with smooth feedback
* Keyboard input support
* Clear and delete functionality

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
```

---

## Getting Started

1. Clone the repository
2. Open the project folder
3. Run `index.html` in any web browser

---

## Usage

Enter numbers and operators using the on-screen buttons or keyboard.
Use the clear button to reset and delete to remove the last input.
